Сравнение функции потерь с регуляризацией и без
1. Без регуляризации
   Функция потерь без регуляризации учитывает только ошибку модели
   в данном случае алгоритм стремится минимизировать только разницу между предсказанными и реальными значениями. Это позволяет быстро уменьшать значение функции потерь, однако при этом веса модели могут становиться слишком большими. В результате модель может переобучаться то есть хорошо работать на обучающих данных, но плохо обобщать новые данные.
2. С регуляризацией
   Функция потерь с L2-регуляризацией имеет дополнительный штраф за величину весов
   Здесь добавляется регуляризационный член(альфа), который ограничивает рост весов модели. Это заставляет модель находить более простые решения.
3. Сравнение поведения
Без регуляризации функция потерь уменьшается быстрее, но модель может переобучаться.
С регуляризацией уменьшение функции потерь происходит медленнее, так как учитывается дополнительный штраф.
Значение функции потерь с регуляризацией обычно больше, чем без неё.
Регуляризация делает модель более устойчивой и снижает риск переобучения.

Регуляризация позволяет контролировать сложность модели за счёт ограничения величины весов. Несмотря на то что значение функции потерь становится выше, модель с регуляризацией обладает лучшей обобщающей способностью и более стабильным поведением.
